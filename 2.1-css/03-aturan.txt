==== Aturan Penulisan ===
Dalam penulisan CSS itu mempunyai aturan tertentu, jika tidak mengikuti aturan diberikan CSS-nya maka Element yang akan kita beri warna misalnya, itu tidak akan berubah sesuai keinginan kita. Mari kita bahas.

Kita ambil contoh berikut:
body {
    background: lightblue;
}

-   Selector
    Selector adalah element HTML yang akan kita pilih dan akan kita ubah style-nya. Seperti pada contoh diatas, kita ingin merubah warna teks tag <body> , maka kita pilih tag <body> sebagai selector-nya lalu diikuti dengan kode untuk merubah warna.
    Apa yg saja yg disebut selector?
    Karna selector bukan hanya tag HTML
    
-   Property
    Property dalam CSS berupa kondisi-kondisi tertentu yang kita tentukan sesuai kebutuhan. Pada contoh diatas "background" lah yang berupa Property, dan masih banyak lagi Property lain-nya yang akan kita bahas juga nanti.

-   Value = Nilai
    Value disini adalah nilai dari property yang sudah kita tentukan sebelumnya. Pada contoh diatas “red” lah value dari property “color”. Artinya kita menentukan warna tag <p> adalah merah.


Kesimpulan :
Untuk merubah style suatu element, kita harus menggunakan 3 aturan diatas. Penjelasan singkat-nya : tentukan tag yang akan kita ubah style-nya, pilih property ( kondisi ) apa yang mau dipakai, lalu tentukan nilainya.
